# Examples

## Example: Candidate FAQ From Resolved Robot Issue

### Input

```markdown
客户问题：
法国客户反馈 CC1 在清洁任务结束阶段出现异常噪音，右侧刮水耙偏移。

产品/机型：
CC1 清洁机器人

飞书知识问答结果：
刮水组件偏移可能导致收尾阶段异响。建议检查刮水组件安装位置、固定螺丝、是否有异物卡住，并确认组件复位。

内部讨论：
技术同事确认视频中噪音来自右侧刮水组件摩擦地面，现场重新安装并固定后噪音消失。

最终解决方案：
重新安装右侧刮水组件，紧固固定点，清理异物，重新执行清洁任务验证通过。
```
